# Planning Records

This folder contains process-generation and semantic-traceback records used by the KGMCF prototype and paper-level process-planning experiments.

Included files:

- `generated_process_plans_raw_400.csv`: method-specific process-plan outputs for held-out cases.
- `process_generation_metrics_raw_400.csv`: process-generation metrics derived from generated plans and KGMCF references.
- `semantic_traceback_records.csv`: symbolic verification and semantic-traceback records.

Only planning and traceback records are included in this project version.
